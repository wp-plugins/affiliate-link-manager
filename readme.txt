Manage all of your affiliate links easily, short links using your own domain, see report of all views and more...

== Description ==

It is a very useful plugin. The Affiliate link Manager plugin allows us manage affiliate links easily. You can...

a) Add new affiliate urls and use a shorten url from your own domain to redirect users. Hidding the affiliate url may grown up your conversions.

b) Delete affiliate urls, of course.

c) See how many views each affiliate url had.

d) See last 1000 access on your affiliate urls, date, time and its origins. 

 

== Installation ==

To install, just follow this steps:

1- Send the plugin to plugins folder, by default it should be: /wp-content/plugins/
2- Do login on your blog's panel, go to Plugins section and activate the plugin Affiliate Link Manager
3- Now, on the left site of your blog's panel, a new option called "Affiliate Link Manager" will appear, click on it to go to the options page
4- Set the options and save changes.
